Pokemon Mystery Dungeon Clone

To recreate Pokemon Mystery Dungeon in Java.

View on GitHub Download .zip Download .tar.gz

Synopsis

This project was started March 28th, 2016, with the ambitious goal of recreating one of the earlier Pokemon Mystery Dungeon games from scratch.

Implemented Features

  • Entity movement
  • Sprite rendering
  • State machine for animating and updating
  • Somewhat messy attack system, including HP and moves
  • Pathfinding - hand-made Djikstra's algorithm
  • Procedural Dungeon Generation
  • Floors and stairs

Graphics

Originally I used Java2D (Swing) to do rendering; now, I've switched to the popular graphics framework libGDX. Sprites are ripped from whatever online sources I could find them from; I do plan to eventually make my own. All rights go to their respective owners.

Progress

I usually spend at least 15 minutes a day working on this project; and I've spent basically the last 8 months working on this. I expect to have a usable release within the next few years at my current pace, but hopefully I will drastically improve by then.
